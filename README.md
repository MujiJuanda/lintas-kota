# Lintas Kota 🚍

## 📖 About

Lintas Kota is a web-based transportation platform designed to simplify intercity travel booking and information services. The application provides users with an easy way to search routes, view schedules, and manage travel bookings through a modern and user-friendly interface.

This project was developed as part of my learning journey in web development and digital service design.

---

## 🎯 Objectives

- Simplify intercity transportation booking.
- Provide travel route and schedule information.
- Improve accessibility to transportation services.
- Deliver a responsive and user-friendly experience.

---

## ✨ Features

- Home Page
- Route Search
- Schedule Information
- Ticket Booking
- User Authentication
- Booking History
- Responsive Design
